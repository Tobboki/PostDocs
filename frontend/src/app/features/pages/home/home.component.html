PostDocs

Free fake REST posts API for testing and prototyping.
Powered by Django + Sqlite .

View on Github Read Docs

Resources

PostDocs comes with a set of 3 common resources:

  • /posts 200 posts
  • /users 15 users
  • /comments 100 comments
  • /photos 500 photos

Note: Resources have relations. For example: users have many posts, ... see Docs for the full list.

Routes

All HTTP methods are supported. You can use http or https for your requests.

  • GET /posts
  • GET /posts/1
  • GET /posts?userId=1
  • GET /comments?postId=1
  • POST

    /posts?userId=1

  • POST

    /comments?postId=1

  • PUT

    /posts

  • PATCH

    /posts/1

  • DELETE

    /posts/1

Note: see Docs for usage example
